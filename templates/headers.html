Gmail

  1. Open the email message

  3. Select "Show original"

Outlook (Web)

  1. Open the email message
  2. Click the three dots (...) in the top right
  3. Select "View message details"
  4. Copy the displayed headers

Outlook (Desktop)

  1. Double-click to open the email in a new window
  2. Go to File > Properties
  3. Find the "Internet headers" section at the bottom
  4. Copy the header text

Apple Mail

  1. Open the email message
  2. Click View > Message > All Headers
  3. Select and copy all header text
